# Poodle

Small, Sharp Scene Tool

## Controls

```
w zoom in
a orbit left
s zoom out
d orbit right
x move up
z move down
r orientation
q center
e export

1 floor
2 ramp
3 wall
4 border

click add
shift click remove
tab toggle grid
del delete last
```
